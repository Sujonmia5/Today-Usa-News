Question 1: what is a difference in var let and const?
Answer: Var let and const difference is:

  • Var declarations can be globally scoped or function scoped, but let and const are only block scoped.
  • Var variables can be updated and redeclared within their scope, let variables can be updated but not redeclared, Const variables cannot be updated or redeclared.
  • All three types of declarations are hoisted above their scope. But while var is given undefined value, let and const are not given any value. Let and const showed error.
  • Var and let can be declared without a value, const must be given a value when declared.

    • Question 3:what is difference map, foreach, filter?

  • Foreach is used to execute the same code on every element in an array but does not change the array and it returns undefined.
  • Map executes the same code on every element in an array and returns a new array with the updated elements.
